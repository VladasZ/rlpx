# rlpx

Implementation of RPLx protocol.

## Dependencies

You need to install Rust and Go.

## Build

Clone repository with submodules:

```bash
git clone --recursive https://github.com/VladasZ/rlpx.git
cd rlpx
```

Build `rplx` library:
```bash
make
```

Build `geth` to start p2p node locally:
```bash
make geth
```
